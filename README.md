# 3D Bus Parking Jam

一個用 HTML5, CSS3 和 Vanilla JavaScript 編寫的 2.5D 益智遊戲。玩家需要透過滑動車輛，將乘客送上對應顏色的公車，清空擁擠的停車場。

## 🎮 遊戲特色

- **2.5D 磁鐵視覺系統**：採用純 CSS 打造的立體效果，兼顧美觀與效能，徹底解決傳統 3D 渲染的破圖問題。
- **動態關卡生成**：內置逆向求解算法，生成 100% 可解的高密度關卡，挑戰無限。
- **智能行駛邏輯**：支援「駛出與轉向 (Pull-out & Turn)」邏輯，模擬真實駕駛軌跡，大幅降低卡死機率。
- **卡通美術風格**：清新明亮的柏油路場景、排隊乘客與具有磁鐵質感的精美車輛。
- **多種色彩匹配**：包含紅、藍、綠、黃、紫 5 種色彩公車，關卡隨機組合。  

## 🚀 快速開始

本項目不需要任何構建工具，直接開啟即可遊玩。

1. 複製倉庫：
   ```bash
   git clone https://github.com/[YOUR_USERNAME]/parking-jam.git
   ```
2. 在瀏覽器中直接開啟 `index.html` 即可開始遊戲。

## 🧠 技術棧

- **Frontend**: HTML5, CSS3 (Custom Properties, Flexbox, Grids, 3D Transforms)
- **Logic**: Vanilla JavaScript (ES6+)
- **Algorithm**: Solver-based Procedural Level Generation (基於求解器的程序化關卡生成)
